# 🏠 Apartment Move-In Checklist

> [!NOTE]
> **Shared checklist — Omar & Mia are both on the lease.** Each item is tagged with an owner:
> **(Both)** = do it together · **(Omar)** / **(Mia)** = one person handles it · **(Each)** = you each do your own copy (IDs, personal accounts).
> Check items off as you go — this doc is live, so whoever does a task can tick the box.

> [!TIP]
> Set a target move date at the top and work backward. A good rhythm is: **2–3 weeks out** = utilities & address changes, **1 week out** = packing & logistics, **move day** = keys + essentials, **first week** = the rest.
>
> **Move date:** _______  ·  **Keys available from:** _______

---

## 1. Set up before moving

Do these in the **2–3 weeks before** the move so nothing lapses on day one.

### Utilities & services (get these ON before move-in day)
- [ ] **(Both)** Confirm which utilities are included in rent vs. your responsibility (check the lease)
- [ ] **(Both)** Set up **electricity** account in your name(s), effective the move-in date
- [ ] **(Both)** Set up **gas** (if separate from electric)
- [ ] **(Both)** Set up **water / sewer / trash** (sometimes landlord-billed — confirm)
- [ ] **(Both)** Order **internet / Wi-Fi** — schedule install for move-in week (book early; slots fill up)
- [ ] **(Either)** Set up / transfer **renters insurance** (many leases require it — get proof before keys)
- [ ] **(Either)** Arrange **home / apartment security** if wanted (or note building's existing system)

### Logistics & planning
- [ ] **(Both)** Book **movers** or reserve a **moving truck / van** (confirm elevator or loading-dock reservation with the building)
- [ ] **(Both)** Confirm building **move-in rules**: elevator booking, allowed hours, certificate of insurance from movers, parking for the truck
- [ ] **(Both)** Do a **change-of-address with the postal service** so mail forwards (see Section 4)
- [ ] **(Each)** Take **time off work** / block the calendar for move day
- [ ] **(Both)** Measure doorways & large furniture — make sure the couch/bed actually fits
- [ ] **(Both)** Order any **furniture / appliances** you need on day one (bed, fridge if unfurnished) with delivery timed to arrival

### Packing
- [ ] **(Both)** Get boxes, tape, bubble wrap, markers
- [ ] **(Both)** **Label boxes by room** + mark "OPEN FIRST" on essentials
- [ ] **(Both)** Pack a **first-night bag** each (see Section 2)
- [ ] **(Both)** Declutter / donate / toss before packing — don't move what you won't keep
- [ ] **(Both)** Photograph electronics/cables before unplugging so reassembly is easy

### Old place wrap-up
- [ ] **(Both)** Give notice / confirm **move-out date** with current landlord
- [ ] **(Both)** Schedule **utility shut-off / transfer** at the old place (don't cancel before you leave)
- [ ] **(Both)** Book move-out cleaning or DIY clean to get the deposit back
- [ ] **(Both)** Document old-place condition with photos for your deposit

## 2. What to bring on move-in day

> [!IMPORTANT]
> Keep these **on you or in the car**, not buried in a moving box. This is the "survive the first 24 hours" kit.

### Documents & keys
- [ ] **(Both)** Signed **lease** copy + landlord/property-manager contact
- [ ] **(Each)** Photo **ID**
- [ ] **(Either)** Proof of **renters insurance** (some buildings won't hand over keys without it)
- [ ] **(Either)** Payment method for any **move-in fees / deposits / first month**

### First-night essentials
- [ ] **(Each)** Overnight bag: change of clothes, toiletries, meds, phone charger
- [ ] **(Both)** Toilet paper, paper towels, hand soap, trash bags
- [ ] **(Both)** Basic **tools**: screwdriver, box cutter, tape measure, flashlight
- [ ] **(Both)** Cleaning supplies (wipes, all-purpose spray, sponge) — clean before unpacking
- [ ] **(Both)** Bedding / air mattress + pillows so you can sleep night one
- [ ] **(Both)** Snacks, water, and a plan for the first meal (nearby takeout is fine)
- [ ] **(Both)** Phone chargers + a **power strip**
- [ ] **(Both)** Light bulbs (in case fixtures are empty)
- [ ] **(Both)** Basic first-aid kit

## 3. When you get the keys

> [!WARNING]
> Do the **walkthrough and photos BEFORE you move a single box in.** This is your evidence for getting the security deposit back later. It's the one thing you can't redo once furniture is in the way.

### Inspect & document (first 30 minutes)
- [ ] **(Both)** Walk through every room and **photo/video document existing damage** (scratches, stains, dents, marks)
- [ ] **(Both)** Fill out the **move-in condition / inventory form** and send it to the landlord in writing (email = timestamp)
- [ ] **(Both)** Test all **outlets, light switches, and fixtures**
- [ ] **(Both)** Run **faucets, shower, toilet** — check water pressure, hot water, and for leaks
- [ ] **(Both)** Test **heating & A/C**
- [ ] **(Both)** Check **appliances** (fridge, stove, oven, dishwasher, washer/dryer) actually work
- [ ] **(Both)** Test **smoke & carbon-monoxide detectors**; note dead batteries
- [ ] **(Both)** Look for **pests** (droppings, traps) and note anything

### Secure & orient
- [ ] **(Both)** **Count and test every key / fob / garage remote** you were given
- [ ] **(Both)** Consider **re-keying or changing locks** (ask landlord first) — you don't know who has old copies
- [ ] **(Both)** Locate the **circuit breaker / fuse box** and label it
- [ ] **(Both)** Locate the **water shut-off valve** and gas shut-off
- [ ] **(Both)** Find **mailbox + mailbox key**; note the unit's mailbox number
- [ ] **(Both)** Confirm **trash / recycling** location and pickup days
- [ ] **(Both)** Get building info: **Wi-Fi for common areas, laundry, parking spot, storage, package handling**
- [ ] **(Both)** Save **emergency contacts**: landlord, super/maintenance, building management

## 4. Update your address everywhere

> [!TIP]
> Start with the **postal forwarding** first — it catches anything you forget. Then work down this list. Many are **(Each)** because you each have your own accounts; shared bills are **(Either)**.

> [!IMPORTANT]
> **Moving to a different state?** The driver's license, vehicle registration, voter registration, and taxes items below come with **legal deadlines** and extra steps. See the companion [Changing States — Complete Move Guide](./changing-states-guide.md) for the full state-switch playbook.

### Government & official
- [ ] **(Each)** **Postal service** change-of-address / mail forwarding
- [ ] **(Each)** **Driver's license / state ID**
- [ ] **(Each)** **Vehicle registration** + title
- [ ] **(Each)** **Voter registration**
- [ ] **(Each)** Tax authority / IRS address on file (or update at next filing)
- [ ] **(Each)** Any **immigration / visa** address requirements, if applicable

### Finances
- [ ] **(Each)** **Banks & credit unions**
- [ ] **(Each)** **Credit cards**
- [ ] **(Each)** Investment / retirement accounts
- [ ] **(Each)** Loans (student, auto, etc.)
- [ ] **(Both)** **Payroll / HR** at work (for W-2s, checks)

### Insurance & health
- [ ] **(Both)** **Renters insurance** (should already reflect the new place)
- [ ] **(Each)** Health / dental / vision insurance
- [ ] **(Each)** Auto insurance (rates can change by address)
- [ ] **(Each)** Life insurance
- [ ] **(Each)** **Doctor, dentist, pharmacy** — transfer prescriptions to a nearby pharmacy

### Subscriptions & everyday
- [ ] **(Each)** Amazon + any shopping accounts (default shipping address)
- [ ] **(Each)** Food delivery / grocery apps
- [ ] **(Each)** Streaming / subscription boxes / meal kits
- [ ] **(Each)** Phone carrier (billing address)
- [ ] **(Both)** Update address on **package lockers / delivery preferences**
- [ ] **(Each)** Loyalty / rewards programs

### People
- [ ] **(Both)** Tell **family & close friends** the new address
- [ ] **(Both)** Update **emergency contact** info you've given to others

## 5. What to cancel or transfer at the old address

> [!CAUTION]
> **Don't cancel too early.** Schedule shut-offs and cancellations for **the day after** your move-out so you're not without power/internet while still packing. And **transfer** rather than cancel where you can — movers, insurers, and carriers often just move the account to the new address.

### Utilities at the old place
- [ ] **(Both)** **Electricity** — schedule final read / shut-off for the day after move-out
- [ ] **(Both)** **Gas** — final read / shut-off
- [ ] **(Both)** **Water / sewer / trash** (if you were billed directly)
- [ ] **(Both)** **Internet / Wi-Fi** — cancel or transfer; **return the router/modem** to avoid equipment fees
- [ ] **(Both)** Cancel any **home security / alarm monitoring** tied to the old unit

### Services & deliveries
- [ ] **(Each)** Cancel or redirect **recurring deliveries** (meal kits, subscription boxes, grocery)
- [ ] **(Each)** Cancel **local memberships** you won't use (gym, pool, parking permit, local storage unit)
- [ ] **(Both)** Cancel **cleaning / lawn / pest / laundry** services scheduled at the old address
- [ ] **(Both)** Stop **newspaper / magazine** or other doorstep deliveries
- [ ] **(Each)** Cancel or transfer **parking spot / permit** at the old building

### Close-out with the old landlord
- [ ] **(Both)** Confirm **move-out date & final rent** owed (prorated?)
- [ ] **(Both)** Do a **move-out walkthrough** and photograph the empty unit for the deposit
- [ ] **(Both)** **Return all keys / fobs / remotes** and get written confirmation
- [ ] **(Both)** Give the landlord your **forwarding address** for the deposit refund
- [ ] **(Both)** Cancel the **old renters insurance** — but only **after** the new policy is active

### Don't forget
- [ ] **(Both)** Empty and clear the **old mailbox**; confirm mail forwarding is live (Section 4)
- [ ] **(Both)** Retrieve anything from **shared storage, garage, or bike room**
- [ ] **(Both)** Return anything **borrowed from neighbors / building** (tools, packages held)

## 6. Other things worth doing

### Safety & security
- [ ] **(Both)** Change or re-key locks (repeat of §3 — genuinely worth it)
- [ ] **(Both)** Note the nearest **hospital, urgent care, pharmacy, and police/fire**
- [ ] **(Both)** Learn the building's **fire exits** and where extinguishers are
- [ ] **(Both)** Set up any **smart-home / security devices** (cameras, doorbell) per lease rules

### Settling in
- [ ] **(Both)** Deep-clean before furniture goes in (kitchen, bathroom especially)
- [ ] **(Both)** Introduce yourselves to **neighbors** and the building super
- [ ] **(Both)** Locate the nearest **grocery store, laundromat, transit stop, gym, hardware store**
- [ ] **(Both)** Set up **recurring bill autopay** for the new utilities so nothing lapses
- [ ] **(Both)** Keep a **running "needs fixing" list** and send it to the landlord in one batch
- [ ] **(Both)** Register for **package / building-access apps** if the building uses them

### Money & records
- [ ] **(Both)** File the **lease, move-in photos, and inventory form** somewhere you'll both find them
- [ ] **(Both)** Track **shared move costs** (deposit, movers, furniture) so you can split fairly
- [ ] **(Both)** Note the **lease end date & renewal window** on a shared calendar
- [ ] **(Both)** Confirm how **rent is paid** and set a reminder a few days before it's due

> [!NOTE]
> **Handy split:** one of you can own **utilities + internet** while the other owns **movers + building logistics**. Address changes are mostly **(Each)** — you each run your own list in Section 4. Do all inspections in Section 3 **together** so you're both witnesses to the condition.
